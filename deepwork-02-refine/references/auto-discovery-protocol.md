# Auto-Discovery — Schnell-Referenz (deepwork-02-refine)

## Wann relevant

Nur wenn beim Section-Planing erkannt wird, dass ein spezifisches Tool für eine Section fehlt (z.B. ein Section-Typ erfordert Figma MCP das nicht installiert ist).

## Ablauf

Kompakter Scan: nur Schritt 3b (MCP-Lücken für den relevanten Section-Typ) ausführen.
