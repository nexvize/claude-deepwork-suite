# Content Protocols — Referenz (deepwork-alt-autopilot)

Im Autopilot-Modus für Content-Projekte: Protokoll-Stack wird aus project-profiles.md geladen.

Swarm-Sizing für Content:
- Einzelner Artikel: 1–2 Worker (Ton-Konsistenz erfordert gemeinsamen Kontext)
- Content-Batch (5–10 Stücke): 3–4 Worker (ein Worker pro Stück + Synthesizer)
- Großes Content-Projekt: 4–6 Worker

Komplette Protokolle: `deepwork-env-setup/references/content-protocols.md`
