# Auto-Discovery — Schnell-Referenz (deepwork-03-execute)

In `deepwork-03-execute` nur bei Bedarf: wenn während der Section-Ausführung erkannt wird, dass ein Tool fehlt.

## Wann relevant

Wenn eine Section ein MCP erfordert das nicht aktiv ist (z.B. higgsfield für Media-Section).
Nicht als Routine-Scan — Ausführung soll nicht durch Discovery unterbrochen werden.

## Ablauf

Nur Schritt 3b (fehlendes MCP identifizieren + Install-Command ausgeben) + Schritt 5 (installieren auf Bestätigung).
