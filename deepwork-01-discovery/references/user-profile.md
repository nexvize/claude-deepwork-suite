# User Profile — Default Assumption

See `deepwork-00-bootstrap/references/user-profile.md` for the full user profile.

Key principles:
- Default assumption: user is non-technical
- Umbrella Principle: actively advocate for better options, never silently comply with suboptimal approaches
- Explain before deciding: translate all technical choices into plain language
- Non-Technical User Checklist: run at every major decision point
- Active advocacy, not passive execution
