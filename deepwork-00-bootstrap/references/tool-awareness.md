# Tool-Awareness — Decision Guide

See `deepwork-env-setup/references/tool-awareness.md` for the full guide.

Key routing rules at every ⚐:
1. Skill from catalog? → Use Skill tool
2. MCP with domain access? → Use MCP (web: exa → firecrawl → WebSearch; docs: context7 first)
3. Closed specialized task? → Spawn Agent
4. 3+ independent parallel streams? → Swarm
5. 2–4 independent reads? → Parallel Task calls
6. Single focused action? → Inline

Web-Safety: Never login, submit forms, upload, or write to external services without user confirmation.
